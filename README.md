# SignInWithApple
SignInWithApple &amp; Using Firebase.
Here, you can user normal Apple login and use with firebase 

## Requirements
- Xcode 10+
- iOS 13.0+

# Firebase 
- Enable Apple Authentication 

# Apple Developer 
- Enable SignIn with Apple in bundle identifier at apple developer console
